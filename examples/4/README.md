# AERIS

Демонстрационный лендинг клубных резиденций, собранный по видеореференсам из `examples/`.

## Запуск

Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

После этого сайт доступен по адресу `http://localhost:8080`.

## Изображения

Активные изображения с суффиксом `-v2.webp` сгенерированы специально для макета и оптимизированы для веба. Исходные кадры из видеореференса оставлены в папке только как материалы анализа и в странице не используются.
